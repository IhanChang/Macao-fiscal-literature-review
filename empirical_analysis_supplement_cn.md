### 1.2 检验1：是否存在稳定的目标配置？

**变异系数（CV）分析（2014-2024）：**

- **USD: CV = 0.186** → ⚠ 中度可变（有一定波动）
- **HKD: CV = 0.236** → ⚠ 中度可变
- **RMB: CV = 0.408** → ✗ **高度可变**（与固定SAA不一致）

**解读标准：** CV < 0.15 表示相对稳定（符合固定SAA）；CV > 0.30 表示高度可变（与严格固定SAA不一致）

**结论：** RMB权重**波动过大**，无法与整个时期的严格固定SAA一致。

### 1.3 检验2：趋势分析

- **USD**: 无显著趋势（斜率 = 0.0035/年，R² = 0.018）
- **HKD**: **显著上升趋势**（斜率 = 0.0125/年，R² = 0.406）⚠️
- **RMB**: 无显著趋势（斜率 = -0.0093/年，R² = 0.099）

**解读：** HKD呈现明显的10年上升趋势（从14.1% → 26.1%），表明**目标权重可能随时间变化**。

### 1.4 检验3：均值回归分析

测试权重偏离均值后是否在次年修正：

- **USD**: 10次观测中8次（80%）表现出均值回归 ✓
- **HKD**: 10次观测中7次（70%）表现出均值回归 ✓
- **RMB**: 10次观测中8次（80%）表现出均值回归 ✓

**解读：** 存在强有力的**均值回归倾向**证据——当权重偏离长期平均值时，倾向于修正。这支持**存在某种形式的目标权重**，尽管不一定是固定的。

### 1.5 检验4：USD-RMB相关性

年度变化的相关系数：**-0.923**（非常强的负相关！）

**解读：** 当USD权重增加时，RMB权重几乎总是减少，反之亦然。这**高度符合替代/再平衡关系**。

### 1.6 检验5：分阶段稳定性分析

**阶段1: 2014-2016（危机期）**
- USD: CV = 0.357，RMB: CV = 0.565（极端波动！）
- 此阶段特征：**主动调整**，而非机械再平衡

**阶段2: 2017-2019（稳定期）**
- USD: CV = 0.127，HKD: CV = 0.131，RMB: CV = 0.146
- 所有货币**高度稳定** —— 符合**固定SAA行为**

**阶段3: 2020-2024（疫后期）**
- USD: CV = 0.108，HKD: CV = 0.083，RMB: CV = 0.097
- **比2017-2019更稳定** —— 表明**SAA框架已确立**

## 二、关键发现：2015-2016年结构性断裂

### 2.1 数据之谜

- 2014年：RMB = 48.6%
- 2015年：RMB = 31.4%（Δ = -17.2pp）
- 2016年：RMB = 13.4%（Δ = -18.0pp）
- **总暴跌：两年内下降35.2个百分点！**

### 2.2 被动估值效应能否解释？

- CNY/USD汇率：6.20（2014）→ 6.49（2015）→ 6.95（2016）
- FX贬值：~-4.5%（2015），~-6.5%（2016）
- 总FX损失：~-11%
- **但权重下降了35pp（相对跌幅72%）！**

**结论：** 仅估值效应**无法**解释这个幅度。必然涉及**主动交易**（抛售RMB资产）。

### 2.3 官方报告的关键证据

**2015年澳门财政储备报告明确指出：**

> "同年财政储备的外汇市场投放录得53.1亿的账面亏损，**主要来自人民币外汇重估损失**。期间，人民银行于八月宣布完善人民币兑美元汇率中间价报价机制，以进一步增强其市场化程度，并**一次性贬值近2%**；至十一月底，国际货币基金组织决定将人民币纳入特别提款权货币篮子，标志着人民币进一步国际化，但其后的海外市场反应令人民币汇价再度受压。**全年总计，在岸人民币下跌4.4%，而离岸人民币则跌近5.4%**。"

**从这一证据中得出的关键洞察：**

1. **确认存在估值损失**：2015年损失53.1亿澳门元
2. **触发事件**：2015年8月"811汇改" - 一次性贬值2%
3. **全年FX损失**：4.4-5.4%
4. 但权重下降**17.2pp** - 估值效应仅能解释约2-3pp

**RMB权重下降的分解：**

```
总变化：-17.2pp（从48.6%降至31.4%）

成分1：被动估值效应
  预期：48.6% × (-5%) ≈ -2.4pp

成分2：主动调整（残差）
  实际 - 预期 = -17.2pp - (-2.4pp) = -14.8pp
```

**这14.8pp的"超额下降"需要估值以外的解释。**

---

### 2.4 三种可能的解释（基于官方证据修订）

**(A) 主动货币去风险（自由裁量的货币配置调整）**
- 811汇改冲击后，金管局主动降低RMB敞口
- 卖出RMB债券/资产（超出自然贬值）
- 买入USD/HKD资产
- 权重下降是**估值和战略配置调整的双重结果**
- → 报告强调"重估损失"暗示这**不是**主要驱动因素

**反对证据**：报告明确将损失归因于"重估"而非"交易"或"投资组合调整"

---

**(B) 资产类别效应（两层决策）⭐ 最可能**

**假说：**
```
2015年前：RMB资产高度集中在风险资产（股票、高收益债券）
811汇改 → 市场压力 → "避险"决策
→ 减少风险资产（股票、高收益债券）
→ 增加安全资产（政府债券、货币市场）

如果：风险资产中70% RMB，安全资产中80% USD
结果：资产类别转移自动创造大规模货币配置调整
```

**为何这能解释幅度：**
- 卖出风险资产自动减少RMB敞口
- 买入安全资产自动增加USD敞口
- 纯估值效应：-2.4pp
- 资产类别效应：-12-14pp（如果RMB高度集中在被卖出的资产中）
- **总计：-14.8pp ✓**

**关键检验：** 2015年报告是否显示：
- 债券配置增加？✓ 需要验证
- 股票配置减少？✓ 需要验证
- 转向更安全工具？✓ 需要验证

**为何报告强调"重估损失"：**
- 这些是持有头寸的按市值计价损失
- 不意味着没有交易，只是损失来自价格变化
- 资产类别配置调整会在资产构成表中单独报告

**可用你的月度资产类别数据检验！**

---

**(C) 综合效应：资产类别 + 战略货币配置调整**

**最现实的情景：**

**阶段1（2015年8-9月）：即时风险管理**
- 估值损失累积（-2.4pp）
- 紧急资产类别配置调整（风险 → 安全）（-5-7pp）

**阶段2（2015年10-12月）：战略重新校准**
- 正式决定降低结构性RMB敞口
- 资产类别内的货币配置调整（-5-7pp）
- 新的隐含目标：RMB ~30%（从~45%下降）

**阶段3（2016年）：持续调整**
- 进一步降至RMB ~13-15%
- 建立新的长期目标

**阶段4（2017年后）：稳定制度**
- 维持RMB目标~20-25%
- 围绕新目标进行有纪律的再平衡

## 三、修订评估：投资组合再平衡机制何时成立？

### 3.1 证据表明：两阶段模型

**阶段1: 2012-2016（战略调整期）**
- **无固定SAA** —— 响应市场条件的大规模战略转移
- 2015-2016年：重大去风险/RMB减持
- 高波动性（USD和RMB的CV > 0.35）
- **机制在此期间不成立**

**阶段2: 2017-2024（机械再平衡期）**
- **相对稳定的SAA**已确立（近似目标：USD ~46%，HKD ~28%，RMB ~24%）
- 低波动性（所有三种货币CV < 0.15）
- 强均值回归（80%的观测）
- 强USD-RMB负相关（-0.92）
- **机制在此期间可能成立**

### 3.2 政策解读

2015-2016年危机（RMB贬值 + 反腐运动）导致了财政储备配置的**战略重新校准**：

- 旧政策（2012-2014）：高RMB敞口（~45%）
- 过渡（2015-2016）：主动减少RMB敞口
- 新政策（2017+）：适度RMB敞口（~24%）并进行有纪律的再平衡

### 3.3 对你研究的启示

1. **不要声称**机械再平衡在2012-2024年全期有效
2. **应该声称**2017年后，证据符合围绕稳定目标的有纪律再平衡
3. **应该承认**2015-2016年涉及战略组合重新校准，而非机械调整
4. **框架应为**："从危机中学习 → 建立有纪律的框架"

## 四、月度数据挑战的三种插值方法

### 4.1 核心问题

我们有年度货币配置权重（2014-2024年年末观测），但需要月度数据来测试高频再平衡响应。问题是：如何在没有实际月度观测的情况下构建月度货币配置序列？

**提议方法：**

使用月度财政储备总额 × 年度货币权重来估算月度各货币持有量。

### 4.2 方法1：年度平均权重 × 月度总额（最简单）

**公式：**
```
货币i金额_t = 财政储备总额_t × 货币i权重_年度平均
```

**2015年例子：**
```
1月RMB金额 = 财政储备总额(2015年1月) × [(48.6% + 31.4%) / 2]
           = 财政储备总额(2015年1月) × 40.0%
```

**优点：**
- ✓ 最容易实施
- ✓ 平滑年末噪音

**严重问题：**
- ❌ **完全丢失时机信息**
- ❌ **2015-2016期间严重偏误**

**2015-2016年的问题示例：**
```
现实可能是：
  1-8月2015：RMB仍为48.6%
  9-12月2015：RMB降至31.4%

使用年度平均（全年40.0%）：
  → 1-8月低估8.6pp
  → 9-12月高估8.6pp
  → 完全错过调整时机
```

**因果分析的致命缺陷：**

如果我们测试"第t月市场压力是否触发再平衡"，但我们人为地平滑了数据以显示全年渐进变化，我们无法检测实际响应时机。

**建议：** ❌ **不要使用此方法** —— 信息损失太大

---

### 4.3 方法2：年末之间线性插值（较好）

**公式：**
```
权重_i,t = 权重_i,t-1_年末 + (权重_i,t_年末 - 权重_i,t-1_年末) × (月份/12)
```

**2015年例子：**
```
6月RMB权重 = 48.6% + (31.4% - 48.6%) × (6/12)
           = 48.6% - 8.6% = 40.0%

然后：6月RMB金额 = 6月财政储备总额 × 40.0%
```

**优点：**
- ✓ 保留全年变化幅度
- ✓ 允许连续变化
- ✓ 数学一致（年末匹配实际数据）

**剩余问题：**
- ⚠️ **假设线性调整**（现实可能是阶跃式）
- ⚠️ **在我们没有信息的地方仍施加结构**

**此方法失效的场景示例：**
```
现实：调整全部发生在2015年8月（811汇改）
  1-7月2015：RMB = 48.6%
  8-12月2015：RMB = 31.4%

线性插值显示月度渐进变化
  → 测试将虚假地发现"调整分布在12个月"
  → 错过集中冲击响应
```

**建议：** ⚠️ **极度谨慎使用**
- 明确标注为"插值，非观测"
- 不要过度解读效应时机
- 用于规模/幅度分析，而非精确时机

---

### 4.4 方法3：年度数据 + 资产类别分解（推荐）⭐

**关键洞见：**

你有**月度资产类别数据**（债券、股票、货币市场工具）。用这些数据来部分打破年度约束！

**两步分解：**

**步骤1：将货币权重变化分解为组成部分**

```
Δ(USD权重)_年度 = [资产类别效应] + [类别内货币效应] + [估值效应]
```

**其中：**
```
资产类别效应 = Δ(债券份额) × (债券中USD - 总体USD权重)
             + Δ(股票份额) × (股票中USD - 总体USD权重)
             + Δ(货币市场份额) × (货币市场中USD - 总体USD权重)
```

**步骤2：使用月度资产类别数据作为月度效应的代理**

由于你观测到月度资产类别配置，你可以构建：
```
隐含USD权重_t = 债券份额_t × [假定债券中USD份额]
               + 股票份额_t × [假定股票中USD份额]
               + 货币市场份额_t × [假定货币市场中USD份额]
```

**需要的关键假设：**
1. 每个资产类别内的货币分布（如"债券是60% USD，30% HKD，10% RMB"）
2. 这些类别内分布是稳定的（或缓慢变化）

**如何估算这些：**
使用年末数据反推隐含的类别内货币分布：
```
2014年年末：
  观测：USD = 28.9%，债券 = 80%
  如果债券中X% USD：需要求解方程组
  使用所有三种货币和三种资产类别 → 9个未知数
  使用多个年末 → 可以估算平均分布
```

**优点：**
- ✓ **使用实际月度变化**（在资产类别中）
- ✓ 可能检测到2015-2016年"避险"转移（股票→债券）
- ✓ 提供经济解释（"USD上升是因为债券配置增加，而非货币转移本身"）

**剩余挑战：**
- 仍需要关于类别内货币分布的假设
- 没有额外信息可能识别不足

**建议：** ✓ **这是最稳健的方法**

**具体实施：**

```r
# 步骤1：使用年末数据估算类别内货币份额
# 求解：观测货币权重 = Σ(资产类别份额 × 货币在资产类别中)

# 步骤2：构建月度隐含权重
月度USD隐含 = 债券_t × 债券中USD +
            股票_t × 股票中USD +
            货币市场_t × 货币市场中USD

# 步骤3：分解年度变化
Δ(USD)_年度 = [债券效应] + [股票效应] + [货币市场效应] + [残差]

# 其中残差 = 类别内货币分布的变化
```

**测试策略：**
```
测试1：[资产类别效应]是否解释大部分2015-2016年RMB下降？
  → 如果是：这是避险，而非货币去风险

测试2：月度资产类别转移是否响应市场压力？
  → 月度Δ(债券份额) = β × VIX + 控制变量

测试3：[残差]（纯货币效应）是否逆周期？
  → 这测试"纯再平衡"假说
```

## 六、修订的实证检验策略

### 6.1 基于实际数据的检验（后2017期仅）

**检验1：估值效应是否触发再平衡？**
```
RMB偏离_t = β₁ · 资本流动_t + 控制变量
```
- **预期**：β₁ < 0（资本外流 → RMB资产权重被动下降）

**检验2：再平衡是否产生外汇损益？**
```
FX损失_t = β₂ · RMB偏离_{t-1} + 控制变量
```
- **预期**：β₂ ≠ 0（权重偏离 → 需要交易纠正 → 产生外汇损益）

**检验3：再平衡是否稳定市场压力？**
```
EMP_t = β₃ · Δlog(财政储备_t) + β₄ · Post2017 · Δlog(财政储备_t) + 控制变量
```
- **预期**：β₄ < 0（再平衡机制运作后，2017年后稳定效应更强）

### 6.2 为何这符合新凯恩斯框架（修订版）

1. 这是**财政操作**（资产管理），而非货币政策干预
2. 不需要央行独立性（只需投资纪律要求）
3. 但通过金融市场流动性产生**准货币稳定效应**
4. 经典的"准财政操作"案例
5. **2017年后**，越来越符合基于规则的行为而非自由裁量

## 七、总结与政策含义

### 7.1 核心发现

1. **2012-2016：战略转型期**
   - 高RMB敞口（~45%）→ 适度RMB敞口（~15%）
   - 主要是响应2015-2016年RMB贬值危机的主动去风险
   - 不能声称机械再平衡

2. **2017-2024：规则性再平衡期**
   - 稳定目标权重（USD ~46%，HKD ~28%，RMB ~24%）
   - 低波动性（CV < 0.15）
   - 强均值回归倾向
   - 符合有纪律的投资组合管理

3. **资产类别效应可能关键**
   - 2015-2016年的巨变可能主要由资产类别转移（股票→债券）驱动
   - 而非纯粹的货币转移
   - 需要用月度资产类别数据验证

### 7.2 对你论文的建议

**应该强调：**
- ✓ 从危机中学习，建立有纪律的框架（2015-2016 → 2017+）
- ✓ 2017年后证据符合准财政稳定机制
- ✓ 通过资产类别数据提供额外验证

**不应该声称：**
- ❌ 整个期间存在严格固定的SAA
- ❌ 所有调整都是机械的
- ❌ 可以精确识别月度再平衡时机
